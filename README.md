# Shell Asia Bot
Shell Account Creator

## Features
- Automatically create shell account, 1 number can used 3 times / 30 minutes
   
## Installation
Requires PHP v7+

```
$ git clone https://github.com/verssache/shellgo
$ cd shellgo
$ php shell.php
```
